# 工具
## 临时评测机

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
	int i = 0;
	while (i < 1000) {
		system("data.exe > data.txt");
		system("baoli.exe < data.txt > baoli.txt");
		system("test.exe < data.txt > test.txt");
		if (system("fc test.txt baoli.txt"))
			break;
		else {
			cout << "success" << endl;
		}
	}
}
```
